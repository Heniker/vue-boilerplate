# Vue 3 + Webpack + Vuetify + Typescript

Type checking is done by [vue-tsc](https://github.com/johnsoncodehk/volar/tree/master/packages/vue-tsc)

### Security
The project uses native Web Crypto API to encrypt and decrypt stored information.<br/>
Provided password is hashed and then resulting hash is used to encrypt and decrypt information.
This is not be a problem because encrypted password is stored locally and is not expected to be changed many times.
Security related functionality is `src/util/security.ts`
